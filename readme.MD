# VCV Rack 2 Patch Analyzer

This Python script analyses a VCV Rack 2
patch file (.vcv) and reports which plugins and
modules are used in the patch. Modules and
plugins are shown in red, if they are not
installed in the local VCV Rack 2 instance.

You must install the Python packages
* pyqt5 and
* zstandard

into the virtual environment for VCVPatchAnalyzer.py.
